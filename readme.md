
# Minimum window substring

Given two strings, find the shortest substring of the first string that contains all letters of the second one. Count matters. Examples:

```
"AAAAABBBB", "BAB" -> "ABB"
"AAABBBCCC", "ABC" -> "ABBBC"
"THELONGTEXT", "ETT" -> "TEXT"
```

# Run

```
npm start
npm test
```
